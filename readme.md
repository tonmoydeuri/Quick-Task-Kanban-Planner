🚀 Features
Drag-and-Drop Interface: Uses the native HTML5 Drag-and-Drop API to move tasks between columns intuitively.

Real-time Task Creation: Instantly generate new tasks with unique IDs using dynamic DOM manipulation.

Responsive Kanban Layout: A modern "Dark Mode" aesthetic built with CSS Flexbox to ensure the board looks great on all screen sizes.

Lightweight & Fast: Built entirely with vanilla JavaScript, eliminating the need for heavy frameworks or libraries.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Transitions)

Logic: Vanilla JavaScript (ES6+)

📂 File Structure
index.html: The core structure of the Kanban board and input systems.

kanban.css: Styling for the board, columns, and task animations.

kanban.js: Logic for task creation and the Drag-and-Drop API implementation.
